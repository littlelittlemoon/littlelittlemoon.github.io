PLA 如何工作

到目前为止，我们讨论了Perceptron如何基于输入信号及其权重做出分类决定。但是Perceptron到底是如何进行学习的，如何找到合适的权重 w0,w1,...,wnw_0, w_1, ..., w_nw0​,w1​,...,wn​，使得感知机模型做出准确的分类？

从 算法1 我们可以看出，感知机学习算法是一种迭代算法，它基于以下简单更新规则：

如果 yi(w⋅xi)<0y_i(w \cdotp x_i) < 0yi​(w⋅xi​)<0 ，更新 www ： - w←w+ηyixiw \gets w+\eta y_ix_iw←w+ηyi​xi​

  • x=[1,x1,...,xn]x = [1, x_1, ..., x_n]x=[1,x1​,...,xn​]
  • w=[w0,w1,...,wn]w = [w_0, w_1, ..., w_n]w=[w0​,w1​,...,wn​]

则 w⋅x=w⊺x=∑i=0nwi∗xiw \cdotp x = w^ \intercal x = \sum_{i=0}^n w_i*x_iw⋅x=w⊺x=∑i=0n​wi​∗xi​。在这里，www和xxx在n+1维空间中只是两个箭头，它们的点积量化了一个向量朝着另一个的方向上的大小。在技​​术上，Perceptron只是在重复的计算点积（在判断 yi(w⋅x)<or>0y_i(w \cdotp x) < \text{or} > 0yi​(w⋅x)<or>0 之前），而真正将正例与负例分开的决策边界是 w⋅x=0w \cdotp x = 0w⋅x=0。

确定决策边界

当我们的数据集线性可分是，通过感知机学习算法我们可以找到能够将正负样本分开的决策边界：w⋅x=0w \cdotp x = 0w⋅x=0。

由向量点积的计算公式：

w⋅x=w⊺x=∥w∥∥x∥cos⁡α w \cdotp x = w^ \intercal x = \parallel w \parallel \parallel x \parallel \cos \alpha w⋅x=w⊺x=∥w∥∥x∥cosα

通过上式，可以得出 α=arccos⁡w⊺x∥w∥∥x∥\alpha = \arccos \frac{w^ \intercal x}{\parallel w \parallel \parallel x \parallel}α=arccos∥w∥∥x∥w⊺x​。故 $w^ x $ 的值存在以下两种情况：

  • w⊺x>0⇒cos⁡α>0⇒α<90w^ \intercal x > 0 \Rightarrow \cos \alpha > 0 \Rightarrow \alpha < 90w⊺x>0⇒cosα>0⇒α<90
  • w⊺x<0⇒cos⁡α<0⇒α>90w^ \intercal x < 0 \Rightarrow \cos \alpha < 0 \Rightarrow \alpha > 90w⊺x<0⇒cosα<0⇒α>90

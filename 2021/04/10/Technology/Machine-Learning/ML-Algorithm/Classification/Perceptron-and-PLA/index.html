感知机模型

感知机称为单层感知机模型，其输入是实例的特征向量，输出为实例的分类类别。它是一种使用阶梯函数激活的人工神经元，以产生二分类输出，用于将数据分为两部分，因此也称为线性二分类器。

感知机的定义

假设输入空间是： X⊆RnX \sube \bold{R}^nX⊆Rn ，输出空间是： Y={+1,−1}Y = \{+1, -1 \}Y={+1,−1} . 输入 x∈Xx \in Xx∈X 表示实例的特征向量，对应于输入空间的点；输出 y∈Yy \in Yy∈Y 表示实例的类别。

  1. 由输入空间到输出空间的函数： f(x)=sign(w⋅x+b)(1) f(x) = sign(w \cdotp x + b) \tag{1} f(x)=sign(w⋅x+b)(1)
    称为感知机；
  2. 其中www和bbb称为感知机模型参数;
  3. w∈Rnw \in \bold{R}^nw∈Rn叫做权值（weight）或权值向量;
  4. b∈Rnb \in \bold{R}^nb∈Rn叫做偏置（bias）;
  5. signsignsign是符号函数： sign(x)={+1x⩾0−1x<0(2) sign(x) = \begin{cases} \tag{2} +1 & x \geqslant 0 \\ -1 & x < 0 \end{cases} sign(x)={+1−1​x⩾0x<0​(2)
  6. 假设空间：是定义在特征空间中的所有线性分类模型或线性分类器： {f∣f(x)=w⋅x+b}\{f|f(x) = w \cdotp x + b \}{f∣f(x)=w⋅x+b} .

感知机学习目标

求出将训练数据进行线性划分的分离超平面。如下图所示的 Decision boundary：

感知机的几何解释：线性方程 w⋅x+b=0w \cdotp x + b = 0w⋅x+b=0 对应于特征空间中的一个超平面 SSS ，其中 www 是超平面的法向量，bbb 是超平民安的截距，这个超平面将特征空间划分为正负两部分：

  1. 当 w⋅xi+b>0w \cdotp x_i + b > 0w⋅xi​+b>0 时，yi=+1y_i = +1yi​=+1;
  2. 当 w⋅xi+b<0w \cdotp x_i + b < 0w⋅xi​+b<0 时，yi=−1y_i = -1yi​=−1.

超平面 SSS 称为分离超平面。求得感知机模型(1)，即求得模型参数 www 和 bbb 。

感知机学习算法（PLA）

学习策略（损失函数）

损失函数的选择有两种方案：

  1. 选择误分类点的总数。这样的损失函数不是关于 www 和 bbb 的连续可导函数，不易优化！（舍弃）
  2. 选择误分类点到超平面的 SSS 的总距离。（✔️采用）

损失函数

根据上述感知机的几何解释，对于误分类点 (xi,yi)(x_i, y_i)(xi​,yi​) 满足：yi(w⋅xi+b)<0y_i(w \cdotp x_i + b) < 0yi​(w⋅xi​+b)<0，假设误分类点的集合为：MMM ，则误分类点到超平面的 SSS 的总距离为：

−1∥w∥∑xi∈Myi(w⋅xi+b) - \frac{1}{\lVert w \rVert} \sum_{x_i \in M} y_i(w \cdotp x_i + b) −∥w∥1​xi​∈M∑​yi​(w⋅xi​+b)

可不考虑 1∥w∥\frac{1}{\lVert w \rVert}∥w∥1​，则损失函数可表示为：

L(w,b)=−∑xi∈Myi(w⋅xi+b) L(w,b) = - \sum_{x_i \in M} y_i(w \cdotp x_i + b) L(w,b)=−xi​∈M∑​yi​(w⋅xi​+b)

即感知机学习的经验风险函数。

最优化问题

感知机学习算法是对以下最优化问题的算法，给定一个训练数据集：

T={(x1,y1),(x2,y2),...,(xN,yN)} T = \{(x_1, y_1), (x_2, y_2), ..., (x_N, y_N)\} T={(x1​,y1​),(x2​,y2​),...,(xN​,yN​)}

其中，xi∈X=Rnx_i \in X = \bold{R}^nxi​∈X=Rn，yi∈Y={+1,−1}y_i \in Y = \{+1, -1 \}yi​∈Y={+1,−1}，i=1,2,...Ni=1,2, ... Ni=1,2,...N，求参数www，bbb，使其为一下损失函数极小化问题的解：

min⁡w,bL(w,b)=−∑xi∈Myi(w⋅xi+b) \min_{w,b} L(w,b) = - \sum_{x_i \in M} y_i(w \cdotp x_i + b) w,bmin​L(w,b)=−xi​∈M∑​yi​(w⋅xi​+b)

其中 MMM 为误分类点的集合。

梯度下降算法极小化损失函数

该损失函数是关于 www 和 bbb 的连续可导函数，那么求极小化问题，很容易想到采用梯度下降法。假设误分类点集合 MMM 是固定的，则损失函数 L(w,b)L(w,b)L(w,b) 的梯度为：

  1. ∇wL(w,b)=−∑xi∈Myixi\nabla_w L(w,b) = -\sum_{x_i \in M} y_ix_i∇w​L(w,b)=−∑xi​∈M​yi​xi​
  2. ∇bL(w,b)=−∑xi∈Myi\nabla_b L(w,b) = -\sum_{x_i \in M} y_i∇b​L(w,b)=−∑xi​∈M​yi​

随机选一个误分类点(x_i, y_i)，对w,bw, bw,b进行更新：

  1. w←w+ηyixiw \gets w+\eta y_ix_iw←w+ηyi​xi​
  2. b←b+ηyib \gets b+\eta y_ib←b+ηyi​

其中 η(0<η⩽1)\eta (0 < \eta \leqslant 1)η(0<η⩽1) 为步长（学习率）。

综上可以得出感知机学习算法的原始形式：

输出：1. w,bw,bw,b；2. 感知机模型：f(x)=sign(w⋅x+b)f(x) = sign(w \cdotp x + b)f(x)=sign(w⋅x+b).

  1. 选取初值 w0,b0w_0, b_0w0​,b0​;
  2. 在训练集中选取数据 (xi,yi)(x_i, y_i)(xi​,yi​)
  3. 如果 yi(w⋅xi+b)<0y_i(w \cdotp x_i + b) < 0yi​(w⋅xi​+b)<0 ，更新 w,bw,bw,b ：
    • w←w+ηyixiw \gets w+\eta y_ix_iw←w+ηyi​xi​

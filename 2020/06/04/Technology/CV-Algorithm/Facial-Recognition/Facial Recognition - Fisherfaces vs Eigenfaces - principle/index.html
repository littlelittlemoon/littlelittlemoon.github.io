人脸识别

广义的人脸识别实际包括构建人脸识别系统的一系列相关技术，包括人脸图像采集、人脸定位、人脸识别预处理、身份确认以及身份查找等；而狭义的人脸识别特指通过人脸进行身份确认或者身份查找的技术。人脸识别是一项热门的计算机技术研究领域，它属于生物特征识别技术，是对生物体（一般特指人）本身的生物特征来区分生物体个体。

面临的挑战

虽然人脸识别有很多其他识别无法比拟的优点，但是它本身也存在许多困难。人脸识别被认为是生物特征识别领域甚至人工智能领域最困难的研究课题之一。人脸识别的困难主要是人脸作为生物特征的特点所带来的。人脸在视觉上的特点是：

  1. 不同个体之间的区别不大，所有的人脸的结构都相似，甚至人脸器官的结构外形都很相似。这样的特点对于利用人脸进行定位是有利的，但是对于利用人脸区分人类个体是不利的。
  2. 人脸的外形很不稳定，人可以通过脸部的变化产生很多表情，而在不同观察角度，人脸的视觉图像也相差很大，另外，人脸识别还受光照条件（例如白天和夜晚，室内和室外等）、人脸的很多遮盖物（例如口罩、墨镜、头发、胡须等）、年龄、拍摄的姿态角度等多方面因素的影响。

在人脸识别中，第一类的变化是应该放大而作为区分个体的标准的，而第二类的变化应该消除，因为它们可以代表同一个个体。通常称第一类变化为类间变化（inter-class difference），而称第二类变化为类内变化（intra-class difference）。对于人脸，类内变化往往大于类间变化，从而使在受类内变化干扰的情况下利用类间变化区分个体变得异常困难。

技术细节

一般来说，人脸识别系统包括图像摄取、人脸定位、图像预处理、以及人脸识别（身份确认或者身份查找）。系统输入一般是一张或者一系列含有未确定身份的人脸图像，以及人脸数据库中的若干已知身份的人脸图象或者相应的编码，而其输出则是一系列相似度得分，表明待识别的人脸的身份。

目前人脸识别的算法可以分类为：

  1. 基于人脸特征点的识别算法（feature-based recognition algorithms）；
  2. 基于整幅人脸图像的识别算法（appearance-based recognition algorithms）；
  3. 基于模板的识别算法（template-based recognition algorithms）；
  4. 利用神经网络进行识别的算法（recognition algorithms using neural network）；
  5. 利用支持向量机进行识别的算法（recognition algorithms using SVM）。

方法

Eigenfaces

特征脸是指一种 appearance-based 的面部识别方法，旨在捕获面部图像集合中的变化并使用此信息对整体中的各个面部图像进行编码和比较（与 parts-based 或 feature-based 的方法相反）的方式。具体而言，特征脸是脸部分布的主要成分，或者等效地，是脸部图像集合的协方差矩阵的特征向量，其中具有N个像素的图像被视为N维空间中的一个点（或向量）。Sirovich和Kirby（Sirovich and Kirby 1987）提出了使用主要成分代表人脸的想法，Turk和Pentland（Turk and Pentland 1991）则采用了这种想法进行人脸检测和识别。Eigenface方法被许多人认为是第一种可行的面部识别技术，并且它是顶级商业面部识别技术产品之一的基础。自从最初的开发和发布以来，自动面部识别系统已经对原始方法进行了许多扩展，并且进行了许多新的开发。特征脸仍然经常被认为是基线比较方法，以证明这种系统的最低预期性能。

特征脸的动机是双重的：

  1. 提取相关的面部信息，该信息可能与人类对眼睛，鼻子和嘴唇等面部特征的直觉没有直接关系。一种方法是捕获面部图像之间的统计差异。
  2. 有效地表示人脸图像。为了减少计算和空间复杂度，可以使用少量参数来表示每个面部图像。

特征脸可以被认为是表征脸部图像之间的整体变化的一组特征。然后，使用特征脸的子集（与最大特征值关联的特征脸）对每个脸图像进行近似。这些特征是训练集中差异最大的。

Fisherfaces

计算机视觉，模式识别和机器学习中的关键问题是为手头的任务定义合适的数据表示。表示输入数据的一种方法是找到一个代表大部分数据差异的子空间。这可以通过使用主成分分析（PCA）来获得。上面我们提到了，当应用于人脸图像时，PCA会生成一组特征脸。这些特征面是与训练数据协方差矩阵的最大特征值相关的特征向量。因此找到的特征向量对应于最小二乘（LS）解。实际上，这是表示数据的一种有效方法，因为它可确保维持数据方差，同时消除样本向量中原始特征（尺寸）之间不必要的现有相关性。当目标是分类而不是特征表示时，LS解决方案可能不会产生最理想的结果。在这种情况下，我们希望找到一种子空间，该子空间将同一类别的样本矢量映射到特征表示的单个点中，而不同类别的样本矢量则尽可能地彼此分开。为实现此目标而衍生的技术称为判别分析（DA）。最著名的DA是线性判别分析（LDA），可以从R.A.提出的想法中得出。1936年的Fisher。当使用LDA查找一组面部图像的子空间表示时，定义该空间的结果基向量称为Fisherfaces。

技术性问题

为了获得Fisherfaces，我们需要计算SwS_wSw​的倒数，即 Sw−1S^{-1}_wSw−1​ 。如果样本特征向量在ppp维空间中定义且 ppp 大于样本总数 nnn ，则 SwS_wSw​ 为奇数。有两种通常使用的解决方案。在第一个解决方案中，我们将样本向量（或等效地，类内和类内散布矩阵）投影到rrr维的PCA空间上，其中 r≤rank(Sw)r ≤ rank(S_w)r≤rank(Sw​) 并在此PCA空间中计算Fisherfaces。第二种解决方案是向SwS_wSw​添加一个正则项。即 Sw+ϵIS_w + ϵISw​+ϵI ，其中 III 是单位矩阵， ϵϵϵ 是一个小常数。

也可以将类间和类内散布矩阵替代做类似工作的其他度量。首先，这两个矩阵是对称的并且是正半定的。因此，每个定义一个度量。这意味着我们可以将这些矩阵替换为其他矩阵，只要它们定义的目标是最小化类内方差和最大化类间距离即可。例如，我们可以将类内散布矩阵替换为样本协方差矩阵。

到目前为止，通过上述方法获得的Fisherfaces基于上述线性假设。当类服从同方差正态时，该假设成立。但通常情况下该假设不成立。要解决此问题，可以在方程式中添加另一个度量。这项新指标的目标是将原始的异方差问题（即具有不同协方差的问题）映射为同方差问题。 借助 Representer's Theorem，该映射函数可以转换为内核映射。并且，度量由Gram（或内核）矩阵给出。因此，这种替代方法称为内核LDA（或简称为KLDA）。几位作者还定义了类内和类间差异的异方差度量。但是，减少正态假设的另一种方法是将每个类别中的样本表示为正态分布的混合。在这种方法中，诀窍是如何确定每个类别的混合物数量。子类判别分析（SDA）算法及其内核扩展KSDA提供了一种流行的解决方案。当训练样本的数量足够大以利于学习非线性映射时，通常首选内核方法。

Fisherfaces

  1. 数据库中人脸之间的差异可能来自诸如照明，面部表情和姿势变化之类的失真。有时，这些变化大于标准面孔之间的变化；
  2. 在变化的光照但固定的姿势下，特定面部的图像位于高维图像空间的3D线性子空间中（无阴影）；
  3. 试图最大程度地减少类内差异，但保留类间差异；
  4. 将图像线性投影到子空间中，该方式可以折衷那些具有较大偏差的面部区域。

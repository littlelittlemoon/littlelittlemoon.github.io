Random Pick with Weight

Given an array w of positive integers, where w[i] describes the weight of index i, write a function pickIndex which randomly picks an index in proportion to its weight.
Note:
1 <= w.length <= 10000
1 <= w[i] <= 10^5
pickIndex will be called at most 10000 times.
Example 1:
Input:
[“Solution”,“pickIndex”]
[[[1]],[]]
Output: [null,0]
Example 2:
Input:
[“Solution”,“pickIndex”,“pickIndex”,“pickIndex”,“pickIndex”,“pickIndex”]
[[[1,3]],[],[],[],[],[]]
Output: [null,0,1,1,1,0]

Solution

有朋友一眼就看懂题目的么？🐻 我是在讨论区看了@logan138的解说才看明白的，哈哈哈(可能是我英语太菜了)。

题目大意是：给定一个权重数组 w，w[i] 代表索引 i 的权重。权重越大，随机获得该索引的概率就越大。假设 w = [1, 3] 。我们可以知道选择每个索引的概率。P(i) = w[i] / 总权重（1 + 3 = 4）因此，对于索引 0: P（0）= 1/4 对于索引1: P（1）= 3/4 因此，有1/4的机会获得索引0和3/4的机会获得索引1。

现在应该都懂了吧，搞懂题意之后，结题思路其实是简单的。我的思路是：我们正常生成随机数，但是由于w[i] >= 1，所以可以把随机数的范围控制在[1, 总权重]范围内。然后根据生成的随机数大小来返回应该返回的index。举个🌰：

  1. w = [1, 3] => total_w = 1 + 3 = 4
  2. 生成随机数：num = random(1, 4) => random num 落在 [1] 则返回index 0，落在 [2, 4] 返回 index 1
  3. 假设 num = 3 => 落在2-4范围 => 返回 index = 1
  4. 假设 num = 1 => 落在1范围 => 返回 index = 0

Code

class Solution:

    def __init__(self, w: List[int]):
        self.w = w
        self.w_len = len(w)

        for i in range(1, self.w_len):
            self.w[i] += self.w[i-1]
        
    def pickIndex(self) -> int:
        num = random.randint(1, self.w[-1])
        
        left = 0
        right = self.w_len - 1 
        
        while left < right:
            mid = (right - left) // 2 + left
            if num > self.w[mid]:
                left = mid + 1
            else:
                right = mid
                
        return left

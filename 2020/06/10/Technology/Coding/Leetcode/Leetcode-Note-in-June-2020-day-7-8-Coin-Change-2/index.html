Kayleen · 2020-6-10 · 次阅读

今天有一道涉及动态规划的题目，这类题目一直以来都是有一定难度的，要多加练习！

Coin Change 2

You are given coins of different denominations and a total amount of money. Write a function to compute the number of combinations that make up that amount. You may assume that you have infinite number of each kind of coin.
Example 1:
Input: amount = 5, coins = [1, 2, 5]
Output: 4
Explanation: there are four ways to make up the amount:
5=5
5=2+2+1
5=2+1+1+1
5=1+1+1+1+1
Example 2:
Input: amount = 3, coins = [2]
Output: 0
Explanation: the amount of 3 cannot be made up just with coins of 2.
Example 3*:
Input: amount = 10, coins = [10]
Output: 1

Solution

🐻 题目大意：给你一个价值amount的币，和一个包含一个或多个价值的币的数组coins，问我们通过用这个数组里面的币组成给定价值amount的币，一共有多少种组合。题意不难理解，但是如何去解这道题还是要多思考一会的。但是在做了很多题之后，我们还是很容易会有一个直觉，这个题从表面上去看，你不可能用蛮力法去解。然后就会很自然的想到，可否用动态规划的思路？ 我们可以思考，怎么将题目要求的问题转为一个一个的小问题，然后通过记录小问题的结果来求得要求问题的结果？

从基本情况没有硬币开始，一一添加硬币。 对于每个添加的硬币，我们从金额 0 到 amount 递归的计算组合数量。

假设dp[i][j]是前i种价值的币可以组成价值为j的币的个数

那么转移方程为: dp[i][j] = dp[i - 1][j] + dp[i][j - coin[i - 1]]

二维数组

class Solution:
    def change(self, amount: int, coins: List[int]) -> int: 
        n = len(coins)
        dp = [[0] * (amount+1) for _ in range(n+1)]

        # 组成价值为0的币只有一种可能，就是不用任何价值的币
        dp[0][0] = 1


        for i in range(1, n + 1):
            # 前面i个组成价值为0的币只有一种情况：不用任何价值的币
            dp[i][0] = 1

            for j in range(1, amount+1):
                dp[i][j] = dp[i-1][j] + (dp[i][j-coins[i-1]] if j >= coins[i-1] else 0)

        return dp[-1][-1]

空间优化

  1. 以基本情况没有硬币开始组合数量。dp[0] = 1，然后其余等于 0。
  2. 遍历所有硬币面值：
    • 对于每个硬币，从金额 0 遍历到 amount：
    • 对于每个 k，计算组合数：dp[k] += dp[k-coin]。
  3. 返回 dp[amount]。
class Solution:
    def change(self, amount: int, coins: List[int]) -> int:
        dp = [0] * (amount + 1)
        dp[0] = 1
        
        for coin in coins:
            for k in range(coin, amount+1):
                dp[k] += dp[k-coin]
                
        return dp[amount]

Power of Two

Given an integer, write a function to determine if it is a power of two.
Example 1:
Input: 1
Output: true
Explanation: 20 = 1
Example 2:
Input: 16
Output: true
Explanation: 24 = 16
Example 3:
Input: 218
Output: false

Solution

这个题很简单，这里主要是记录一下解法，怕自己以后忘了。

我一开始的思路是转成二进制，然后看1的个数。后面发现可以直接用位运算&，但是我没想到，所以记录一下。

class Solution:
    def isPowerOfTwo(self, n: int) -> bool:
#         n_bin = str(bin(n))[2:]
#         counter_n_bin = Counter(n_bin)
        
#         return n > 0 and counter_n_bin['1'] == 1

        return n > 0 and not (n & n-1)

Count Square Submatrices with All Ones

Given a m * n matrix of ones and zeros, return how many square submatrices have all ones.
Example 1:
Input: matrix =
[
[0,1,1,1],
[1,1,1,1],
[0,1,1,1]
]
Output: 15
Explanation:
There are 10 squares of side 1.
There are 4 squares of side 2.
There is 1 square of side 3.
Total number of squares = 10 + 4 + 1 = 15.
Example 2:
Input: matrix =
[
[1,0,1],
[1,1,0],
[1,1,0]
]
Output: 7
Explanation:
There are 6 squares of side 1.
There is 1 square of side 2.
Total number of squares = 6 + 1 = 7.

Solutions

这个题和之前做过的那道求最大正方形面积的题很像，只不过这道题求的是正方形的个数。还是一样的思路，采用动态规划，这里再复习一下好了。

思路是扫描矩阵中的每个单元格，以使用可以从当前单元格（当它是任何可能的正方形的右下角单元格）形成的正方形数来更新矩阵单元格的变量值为当前单元格可形成的正方形个数。说起来可能比较空洞，这里引用@arkaung的图解，他真的做的很棒！

class Solution:
    def countSquares(self, matrix: List[List[int]]) -> int:
        m, n = len(matrix), len(matrix[0])
        count = 0
        
        for i in range(m):
            for j in range(n):
                if i != 0 and j != 0:
                    # 当i和j都非零时，由于矩阵中只存在0和1，所以遍历到当前点时，
                    matrix[i][j] *= min(matrix[i-1][j], matrix[i][j-1], matrix[i-1][j-1]) + 1

                # 计算当前总共累计的正方形个数  
                count += matrix[i][j]
                
        return count

References: 上图源自 [@arkaung's discussion](https://leetcode.com/explore/challenge/card/may-leetcoding-challenge/536/week-3-may-15th-may-21st/3336/discuss/643429/Python-DP-Solution-+-Thinking-Process-Diagrams-(O(mn)-runtime-O(1)-space))

Interval List Intersections

Given two lists of closed intervals, each list of intervals is pairwise disjoint and in sorted order.
Return the intersection of these two interval lists.
(Formally, a closed interval [a, b] (with a <= b) denotes the set of real numbers x with a <= x <= b. The intersection of two closed intervals is a set of real numbers that is either empty, or can be represented as a closed interval. For example, the intersection of [1, 3] and [2, 4] is [2, 3].)
Example:

Input: A = [[0,2],[5,10],[13,23],[24,25]], B = [[1,5],[8,12],[15,24],[25,26]]
Output: [[1,2],[5,5],[8,10],[15,23],[24,24],[25,25]]

Solution

从上面举例中的图可以很直观的知道题目要求。我做这道题的思路是使用两个指针分别指向A，B两个数组，根据间隔交集的特点来找交集。交集的特点也很明显：交集的上界只可能产生于A或B中当前间隔的上界，下界只能产生于A或B中当前间隔的下界。具体看代码实现。

class Solution:
    def intervalIntersection(self, A: List[List[int]], B: List[List[int]]) -> List[List[int]]:
        len_a = len(A)
        len_b = len(B)
        inter = []
        a_i = b_i = 0

        while a_i < len_a and b_i < len_b:
            [a_0, a_1], [b_0, b_1] = A[a_i], B[b_i] 
            if a_0 <= b_1 and b_0 <= a_1:
                inter.append([max(a_0, b_0), min(a_1, b_1)])
            
            if a_1 <= b_1:
                a_i += 1
            else:
                b_i += 1
                
        return inter

Construct Binary Search Tree from Preorder Traversal

Return the root node of a binary search tree that matches the given preorder traversal.
(Recall that a binary search tree is a binary tree where for every node, any descendant of node.left has a value < node.val, and any descendant of node.right has a value > node.val. Also recall that a preorder traversal displays the value of the node first, then traverses node.left, then traverses node.right.)
It's guaranteed that for the given test cases there is always possible to find a binary search tree with the given requirements.
Example 1:
Input: [8,5,1,7,10,12]
Output: [8,5,10,1,7,null,12]

Solution

这题就是要求我们通过题目给定的数组，构造一个搜索二叉树。我最开始的思路就是我本科数据结构学的构造搜索二叉树的算法(学没白上😋。可以通过遍历数组，依次根据搜索二叉树的特点（左子树节点的值小于右子树节点的值）向树中插入新的节点，使新的节点插入后也满足二叉搜索树的特点。

Mine

# Definition for a binary tree node.
# class TreeNode:
#     def __init__(self, val=0, left=None, right=None):
#         self.val = val
#         self.left = left
#         self.right = right

class Solution:
    def bstFromPreorder(self, preorder: List[int]) -> TreeNode:
        def bstInsert(tree, value):
            if not(tree):
                tree = TreeNode()
                tree.val = value
                return tree
            
            elif tree.val > value:
                tree.left = bstInsert(tree.left, value)
                return tree
            else:
                tree.right = bstInsert(tree.right, value)
                return tree

        tree = None
        for value in preorder:
            tree = bstInsert(tree, value)
            
        return tree

Nice answer from @ekovalyov

我再讨论区看到了一个很nice的答案，来自@ekovalyov，这里马一下。

class Solution:
    def bstFromPreorder(self, preorder: List[int]) -> TreeNode:
        if preorder:
            return TreeNode(preorder[0],
                            self.bstFromPreorder(tuple(takewhile(lambda x:x<preorder[0], preorder[1:]))),
                            self.bstFromPreorder(tuple(dropwhile(lambda x:x<preorder[0], preorder[1:]))))

Course Schedule

There are a total of numCourses courses you have to take, labeled from 0 to numCourses-1.
Some courses may have prerequisites, for example to take course 0 you have to first take course 1, which is expressed as a pair: [0,1]
Given the total number of courses and a list of prerequisite pairs, is it possible for you to finish all courses?
Example 1:
Input: numCourses = 2, prerequisites = [[1,0]]
Output: true
Explanation: There are a total of 2 courses to take. To take course 1 you should have finished course 0. So it is possible.
Example 2:
Input: numCourses = 2, prerequisites = [[1,0],[0,1]]
Output: false
Explanation: There are a total of 2 courses to take. To take course 1 you should have finished course 0, and to take course 0 you should also have finished course 1. So it is impossible.

Solution

这个题目大意是根据先行课要求，判断能不能完成所有课程。透过题目我们可以看到这题本质上是一个判断有向图是否存在环的问题。如果存在环那么不能完成课程，不存在环则可顺利完成所有课程。判断有向图是否存在环可以采用深度优先算法遍历图。

class Solution:
    def canFinish(self, numCourses: int, prerequisites: List[List[int]]) -> bool:
        neighbours = [[] for x in range(numCourses)]
        
        for v, u in prerequisites:
            neighbours[u].append(v)
            
        for source in range(numCourses):
            visited = [False for x in range(numCourses)]
            s = [source]

            # DFS
            while s:
                tmp = s.pop()
                if not visited[tmp]:
                    s.extend(neighbours[tmp])
                    
                # Cycle
                elif tmp == source:
                    return False
                
                visited[tmp] = True
                
        return True

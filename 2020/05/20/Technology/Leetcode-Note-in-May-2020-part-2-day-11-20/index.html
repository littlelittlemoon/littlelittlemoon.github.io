Remove K Digits

Given a non-negative integer num represented as a string, remove k digits from the number so that the new number is the smallest possible.
Note:
The length of num is less than 10002 and will be ≥ k.
The given num does not contain any leading zero.
Example 1:
Input: num = "1432219", k = 3
Output: "1219"
Explanation: Remove the three digits 4, 3, and 2 to form the new number 1219 which is the smallest.
Example 2:
Input: num = "10200", k = 1
Output: "200"
Explanation: Remove the leading 1 and the number is 200. Note that the output must not contain leading zeroes.
Example 3:
Input: num = "10", k = 2
Output: "0"
Explanation: Remove all the digits from the number and it is left with nothing which is 0.

Solutions

Mine(贪心算法)

做了有点久了，还有点想不起来了，以后还是做一道记录一道比较好。这题的大概意思是：从给定的数字字符串中，删除k个数字，使得删除后的数字最小。这个题我自己做的时候是思路是，从左到右每次选取最小的数字，作为删除k个数字后的对应位的数字。感觉说的不是很清楚。可以简单的看一个例子：
num = '1432219', k = 3，意味着要从num中删除3个数字，使剩下的4个数字组成的数最小，且数字的相对位置不可以改变。删除3个数字还剩4个，那么假设剩余四个数字为：ABCD， 那么A只能从'1432'中选一个，为了保证最后的数最小，所以这里选最小的：1。同理B只能从'4322'（1已经被选了）中选最小的：2，C：1，D：9。所以最后的Output应该是：'1219'。

思路就是这样，具体看看代码实现。

class Solution:
    def removeKdigits(self, num: str, k: int) -> str:
        n = len(num)
        
        if n<= k : return '0'
        
        t = s = 0
        res = ''
        first_none_zero = False
        
        for e in range(k, n):
            tmp = num[t]
            # 从左到右，找最小数字对应位（注意在原数字字符串的可选区间）的最小数字
            for i, n in enumerate(num[s:e+1]):
                if n < tmp:
                    t, tmp = s+i, n
            t += 1
            s = t
            if tmp != '0' : first_none_zero = True
                
            if first_none_zero: res += tmp
            
        return res or '0'

使用正则表达式

我在做完之后去看了一下其他小伙伴的讨论，我发现大部分小伙伴的思路都和我那个大同小异。也遇到了其他的解决方案，就是用正则表达式，我觉得这个挺厉害的，我也没想到，学习一下，具体看代码，如果看不懂的话可能要去看看正则表达式的相关内容。

class Solution:
    def removeKdigits(self, num: str, k: int) -> str:
        sub = re.compile('1[0]|2[01]|3[0-2]|4[0-3]|5[0-4]|6[0-5]|7[0-6]|8[0-7]|9[0-8]|.$').sub

        for _ in range(k):
            num = sub(lambda m: m.group()[1:], num, 1)
        
        return num.lstrip('0') or '0'

Online Stock Span

Write a class StockSpanner which collects daily price quotes for some stock, and returns the span of that stock's price for the current day.
The span of the stock's price today is defined as the maximum number of consecutive days (starting from today and going backwards) for which the price of the stock was less than or equal to today's price.
For example, if the price of a stock over the next 7 days were [100, 80, 60, 70, 60, 75, 85], then the stock spans would be [1, 1, 1, 2, 1, 4, 6].
Example 1:
Input: ["StockSpanner","next","next","next","next","next","next","next"], [[],[100],[80],[60],[70],[60],[75],[85]]
Output: [null,1,1,1,2,1,4,6]
Explanation:
First, S = StockSpanner() is initialized. Then:
S.next(100) is called and returns 1,
S.next(80) is called and returns 1,
S.next(60) is called and returns 1,
S.next(70) is called and returns 2,
S.next(60) is called and returns 1,
S.next(75) is called and returns 4,
S.next(85) is called and returns 6.
Note that (for example) S.next(75) returned 4, because the last 4 prices
(including today's price of 75) were less than or equal to today's price.

Solution

这个题蛮有意思的，题目大意是：寻找股票价格的跨度，也就是找之前（包括当前价格）有连续有多少天的价格不高于今天的价格。我感觉我不会说话了，哈哈哈，其实看上面那个例子就很清楚了。这里我的思路也是常规思路，就是按照题意每天新价格参产生的时候，记录当前价格的跨度。然后有一个优化方案，我们不必记录所有价格的跨度，因为有的是可以累加(例如同小于当前价格的跨度可以累加作为当前价格的跨度)。具体优化看代码实现。

class StockSpanner:

    def __init__(self):
        self.stack = []

    def next(self, price: int) -> int:
        cur = 1
        while self.stack and self.stack[-1][0] <= price:
            cur += self.stack.pop()[1]
            
        self.stack.append((price, cur))
        
        return cur

Kth Smallest Element in a BST

Given a binary search tree, write a function kthSmallest to find the kth smallest element in it.
Example 1:
Input: root = [3,1,4,null,2], k = 1

  3
 / \
1   4
 \
  2

Output: 1

Example 2:
Input: root = [5,3,6,2,4,null,null,1], k = 3

      5
     / \
   3   6
   / \
  2   4
 /
1

Output: 3

Follow up:
What if the BST is modified (insert/delete operations) often and you need to find the kth smallest frequently? How would you optimize the kthSmallest routine?

Solution

题目大意：找二叉搜索树中第k小的节点。我的思路很直接，就是根据二叉排序树的特点（左子树小于右子树的节点），采用深度优先算法遍历二叉树，然后依次将左节点值(如果存在)和右节点值(如果存在)放入一个列表里，根据二叉搜索树的特点，这样生成的列表是从小打到排序的。最后直接返回列表中第k个值就好了。具体看代码实现。

# Definition for a binary tree node.
# class TreeNode:
#     def __init__(self, val=0, left=None, right=None):
#         self.val = val
#         self.left = left
#         self.right = right

class Solution:
    def kthSmallest(self, root: TreeNode, k: int) -> int:
        if not root:
            return None
        
        def dfs(node):
            if not node:
                return None
            
            dfs(node.left)
            sorted_list.append(node.val)
            dfs(node.right)
            
        sorted_list = []
        dfs(root)
        
        return sorted_list[k-1]

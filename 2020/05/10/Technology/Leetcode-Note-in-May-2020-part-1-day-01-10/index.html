Majority Element

Given an array of size n, find the majority element. The majority element is the element that appears more than ⌊ n/2 ⌋ times.
You may assume that the array is non-empty and the majority element always exist in the array.
Example 1:
Input: [3,2,3]
Output: 3
Example 2:
Input: [2,2,1,1,1,2,2]
Output: 2

Solution

这个题不难的，这里记录它是因为可以学到一个Python的collection，直接上代码。

class Solution:
    def majorityElement(self, nums: List[int]) -> int:
        return Counter(nums).most_common(1)[0][0]

哈哈哈，这代码不要太简洁，舒服~

Cousins in Binary Tree

In a binary tree, the root node is at depth 0, and children of each depth k node are at depth k+1.
Two nodes of a binary tree are cousins if they have the same depth, but have different parents.
We are given the root of a binary tree with unique values, and the values x and y of two different nodes in the tree.
Return true if and only if the nodes corresponding to the values x and y are cousins.
Example 1:
Input: root = [1,2,3,4], x = 4, y = 3

    1  
   / \  
  2   3

Output: false

Example 2:
Input: root = [1,2,3,null,4,null,5], x = 5, y = 4

  -1
  /  \
 2    3
  \    \
   4    5

Output: true

Example 3:
Input: root = [1,2,3,null,4], x = 2, y = 3

   1
  /  \
 2    3
  \    
   4

Output: false

Solution

这个题是判断给定的两个节点，在二叉树中是否为堂兄弟，堂兄弟的定义是：在树的同一层，但父节点不同，我这里一开始想到的方法就是用层次遍历算法，然后注意判断条件就行，具体diamante注释都写得很清楚了。

# Definition for a binary tree node.
class TreeNode:
    def __init__(self, val=0, left=None, right=None):
        self.val = val
        self.left = left
        self.right = right
        
class Solution:
    def isCousins(self, root: TreeNode, x: int, y: int) -> bool:
        q = []
        q.append(root)
        
        # 层次遍历二叉树：通过判断：
        # 1. x，y节点是否在同一层
        # 2. x，y节点的父节点是否相同
        # 来判断x，y是否为堂兄弟
        while q:
            tmp, n = [], len(q)
            for i in range(n):
                cur = q.pop(0)
                if cur:
                    tmp.append(cur.val)
                    q.append(cur.left)
                    q.append(cur.right)
                else:
                    # 加None占位, 方便判断是否为同一父节点
                    tmp.append(None)
                    
            # 如果x，y在同一层，则可能是堂兄弟
            if x in tmp and y in tmp:
                l = tmp.index(x)
                r = tmp.index(y)
                if l > r:           
                    l, r = r, l
                    
                # 1. 如果l, r相邻（r-l == 1）并且r为奇数（r%2 != 0）
                # （l为偶数），则父节点相同 => 非堂兄弟
                if r-l == 1 and r%2 != 0:     
                    return False
                
                # 2. 同层且父节点不同 => 堂兄弟
                return True
            
            # 3. 如果x，y不在同一层 => 非堂兄弟
            if x in tmp or y in tmp:
                return False
            
        # 4. x, y都不在二叉树中
        return False

Valid Perfect Square

Given a positive integer num, write a function which returns True if num is a perfect square else False.
Note: Do not use any built-in library function such as sqrt.
Example 1:
Input: 16
Output: true
Example 2:
Input: 14
Output: false

Solution

这个题我没有什么好的思路，然后看到讨论区有一个不错的方法，我觉得可以拿来参考参考。

这个解法用的牛顿法：可用于快速近似平方根（二次收敛，因此正确数字的数量每次迭代加倍）

x0≈Sxn+1=12(xn+Sxn)S=lim⁡n→∞xn x_0 \approx \sqrt{S} \\ x_{n+1} = \frac{1}{2} \Big(x_n + \frac{S}{x_n} \Big) \\ \sqrt{S} = \lim\limits_{n\rightarrow\infty}x_n x0​≈S​xn+1​=21​(xn​+xn​S​)S​=n→∞lim​xn​

class Solution:
    def isPerfectSquare(self, num: int) -> bool:
        r = num
        while r*r > num:
            r = (r + num/r) // 2
        return r*r == num

Find the Town Judge

In a town, there are N people labelled from 1 to N. There is a rumor that one of these people is secretly the town judge.
If the town judge exists, then:
1. The town judge trusts nobody.
2. Everybody (except for the town judge) trusts the town judge.
3. There is exactly one person that satisfies properties 1 and 2.
You are given trust, an array of pairs trust[i] = [a, b] representing that the person labelled a trusts the person labelled b.
If the town judge exists and can be identified, return the label of the town judge. Otherwise, return -1.
Example 1:
Input: N = 2, trust = [[1,2]]
Output: 2
Example 2:
Input: N = 3, trust = [[1,3],[2,3]]
Output: 3
Example 3:
Input: N = 3, trust = [[1,3],[2,3],[3,1]]
Output: -1
Example 4:
Input: N = 3, trust = [[1,2],[2,3]] Output: -1
Example 5:
Input: N = 4, trust = [[1,3],[1,4],[2,3],[2,4],[4,3]]
Output: 3
Note:
1. 1 <= N <= 1000
2. trust.length <= 10000
3. trust[i] are all different
4. trust[i][0] != trust[i][1]
5. 1 <= trust[i][0], trust[i][1] <= N

Solution

这个题主要是想熟悉一下set和Counter的用法，所以我想方设法也要用到Counter和set。注释写得很清楚，直接看代码吧。

class Solution:
    def findJudge(self, N: int, trust: List[List[int]]) -> int:
        # 特判：
        # 1. 如果谁也不相信谁，且人数大于1，则一定没有法官
        # 2. 如果只有一个人，则这个人是法官
        if not(trust):
            if N > 1: return -1
            return N
        
        n_judge = set([t[0] for t in trust])
        c_judge = set([t[1] for t in trust])
        
        # 可能出现的法官：被相信，但不相信任何人
        #（存在被相信的人群中，但不存在相信别人的人群中）
        tmp_judge = c_judge.difference(n_judge)
        
        # 如果没有满足上诉条件的人，则法官不存在
        if not(tmp_judge): return -1
        
        # 如果存在这样的人，由题意，只存在一个这样的人
        # 再判断：是否除他自己以外其他人都相信他
        if Counter([t[1] for t in trust]).most_common(1)[0][1] == N-1:
            return tmp_judge.pop()
        
        return -1
